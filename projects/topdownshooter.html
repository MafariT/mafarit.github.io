About the TopDownShooter Project

Cool blocky-style game developed using Godot game engine

Features

Currently still in the alpha development, so the game only has a few features like basic enemy AI, enemy deaths and firing sounds. It only has one level at the moment but still fun to play!

Will add more features in the future:
